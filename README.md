## Dependencies
- [NodeJS / Treesitter](https://nodejs.org/en/download)
- [Python](https://www.python.org/)
- [TDM-GCC (Windows)](https://jmeubank.github.io/tdm-gcc/)
- [Git](https://git-scm.com/install/)
- [.NET](https://dotnet.microsoft.com/pt-br/download)
- [Nerd Font](https://www.nerdfonts.com/)
- [Gnu Make](https://gnuwin32.sourceforge.net/packages/make.htm)

---

## Clone the repository

#### Linux
```shell
git clone https://github.com/joaostack/stacknvim ~/.config/nvim
```

#### Windows Powershell
```shell
git clone https://github.com/joaostack/stacknvim $env:LOCALAPPDATA/nvim
```

## Start Neovim

```shell
nvim
```


#### Update
```
:Lazy sync
```
