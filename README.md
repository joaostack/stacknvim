#### Clone the repository

#### Linux
```shell
git clone https://github.com/joaostack/stacknvim ~/.config/nvim
```

#### Windows Powershell
```shell
git clone https://github.com/joaostack/stacknvim $env:LOCALAPPDATA/nvim
```

#### Start Neovim

```shell
nvim
```

#### Install C# Roslyn LSP
```
MasonInstall roslyn
```
